# Union Shop

This is a Flutter project for an e-commerce application called "Union Shop".

## Navigation

To navigate from the Clothing page to the Collection page, follow these steps:

1.  Click on the "Shop" menu button.
2.  From the dropdown menu, select "Clothing" to go to the Clothing page.
3.  Click on the "Shop" menu button again.
4.  From the dropdown menu, select "Collection" to go to the Collection page.

## Buying a Product

To buy a product, follow these steps:

1.  On the product list, click the "Add to Cart" button for the desired product.
2.  This will add the product to your cart and take you to the cart page.
3.  On the cart page, you can review your order and proceed to checkout.
